# Calculator Project

## 개요
이 프로젝트는 Main Camp 2주차에 개발된 간단한 계산기 애플리케이션입니다. 이 계산기는 덧셈, 뺄셈, 곱셈, 나눗셈과 같은 기본적인 산술 연산을 지원합니다.

## 개발기간
2025.2.26 ~ 2025.3.5

## 기능
- 덧셈
- 뺄셈
- 곱셈
- 나눗셈

## 사용법
계산기를 사용하려면 원하는 산술 연산과 계산하려는 숫자를 입력합니다. 결과가 화면에 표시됩니다.

## lv 별 특징
- Lv1
    - 다른 클래스 없이 Main 클래스에서만 작동하는 프로그램
- Lv2
    - Calculator 클래스를 통해 Lv1의 사칙연산 부분 분리
    - Index 클래스를 통해 사칙 연산 후 결과를 저장하고 저장된 결과 삭제
    - try-catch를 통한 예외처리
- Lv3
    - enum 클래스 OperatorType를 통한 연산자 정보 관리
