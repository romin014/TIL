## 2. GROUP BY
* GROUP BY
    * 특정 열을 기준으로 데이터를 그룹화해주는 문법

* GROUP BY 의 구조
```
select 카테고리 컬럼,  
        sum(계산 컬럼) 별명
from 테이블
group by 카테고리 컬럼
```
사용예시
>select major, avg(age) avg_age\
from student  
group by major

## 3. ORDER BY
* ORDER BY
    * select 문에서 나온 결과를 정렬하기 위한 문법

* ORDER BY 의 구조
```
select 카테고리 컬럼,
        sum(계산 컬럼)
from 테이블
group by 카테고리 컬럼
order by 정렬을 원하는 컬럼
* asc : 오름차순
* desc : 내림차순
```
사용예시
>select major,
        avg(age) avg_age\
from student  
order by major desc
